# map-extra

[![NPM](https://nodei.co/npm/map-extra.png)](https://nodei.co/npm/map-extra/)

Extra methods for [map].<br>
Browserified, minified module: [map-extra.min].

```javascript
const extra = require('map-extra');
// -> updated Map constructor
// Map.is(<value>)
// Map.equal(<map1>, <map2>)
// ...
```
